# 🟢 Sanemi

## Effets

{% hint style="success" %}
Vous disposez des effets <mark style="color:blue;">**Speed 1**</mark> le jour et <mark style="color:blue;">**Speed 2**</mark> la nuit.

Vous possédez une [<mark style="color:green;">**Lame de Nichirin**</mark>](./#lame-de-nichirin).
{% endhint %}
